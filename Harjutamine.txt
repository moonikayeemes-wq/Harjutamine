GitHub on veebipõhine platvorm, mida kasutatakse koodi ja muude digitaalsete projektide hoidmiseks, haldamiseks ja ühiseks arendamiseks. See võimaldab inimestel üle maailma koos töötada, jälgida tehtud muudatusi ning arendada tarkvara organiseeritult ja turvaliselt. GitHubi kasutavad peamiselt programmeerijad, kuid ka ettevõtted, õpilased, õpetajad ja avatud lähtekoodiga kogukonnad, et jagada oma tööd, õppida üksteiselt ning luua kvaliteetseid digilahendusi.
Mina saan edaspidi kasutada GitHubi oma koolitööde ja programmeerimisprojektide salvestamiseks ning versioonide haldamiseks. Samuti aitab GitHub mul teha koostööd teistega ja hoida oma tööd korrastatuna.
GitHub asutati 2008. aastal ettevõttena GitHub Inc, mille asutasid Tom Preston-Werner, Chris Wanstrath ja PJ Hyett. GitHubi teenus muutus avalikult kättesaadavaks samuti 2008. aastal ning on sellest ajast kasvanud üheks populaarseimaks arendusplatvormiks maailmas.
